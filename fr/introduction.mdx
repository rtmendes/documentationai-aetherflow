## Découvrir AetherFlow

AetherFlow aide votre équipe à automatiser des processus métiers, connecter vos outils entre eux et orchestrer des workflows de bout en bout, sans perdre en contrôle ni en visibilité.

Avec AetherFlow, vous définissez des workflows composés de déclencheurs, d étapes et d intégrations vers vos outils existants (messagerie, CRM, paiement, systèmes internes). La plateforme exécute ensuite ces workflows de façon fiable et traçable.

## Ce que vous trouverez dans cette documentation

Cette documentation vous guide pour :

- Comprendre les concepts clés des workflows AetherFlow.
- Mettre en place vos premiers scénarios d automatisation d équipe.
- Intégrer AetherFlow à vos outils existants (Slack, Microsoft Teams, Discord et autres services).
- Utiliser les APIs, webhooks et mécanismes d authentification sécurisée.
- Surveiller, diagnostiquer et optimiser vos automatisations dans le temps.

Vous pouvez parcourir la documentation de manière linéaire pour un premier déploiement, ou accéder directement aux sections dont vous avez besoin lorsque vous concevez ou dépannez un workflow.

<Callout kind="info">

### Avant de commencer

Assurez vous d avoir :

- Un compte AetherFlow d équipe actif.
- Les droits nécessaires pour créer ou modifier des workflows.
- Une clé API ou un mécanisme d authentification configuré si vous prévoyez d utiliser l API ou les webhooks.

Si votre organisation applique des règles de sécurité strictes, coordonnez vous avec votre équipe sécurité avant de connecter de nouveaux services.

</Callout>

## Comment utiliser ces docs en équipe

Pour un premier contact avec AetherFlow, commencez par les bases, créez un petit workflow de test, puis élargissez progressivement à des cas d usage plus critiques.

Les profils techniques (développeurs, ingénieurs plateforme) trouveront les détails d intégration, d API et de sécurité dans les sections spécialisées. Les profils métiers (ops, support, produit) peuvent se concentrer sur la conception des workflows, les bonnes pratiques et le suivi opérationnel.
